# Number Snake 게임 기획서

## 컨셉 요약

스네이크를 조작해 숫자 먹이를 먹고, 같은 숫자끼리 합쳐서 점수를 올리는 퍼즐 아케이드 게임.
스네이크 조작 + 수박게임 합체 쾌감 + 2048 숫자 전략이 결합된 구조.

---

## 핵심 규칙

### 이동
- 스와이프 또는 방향키로 뱀을 조작
- 벽에 닿으면 반대편으로 **wrap** (죽지 않음)

### 먹기
- 머리 숫자 **이하**의 먹이만 먹을 수 있음 → 꼬리 끝에 붙어서 성장
- 머리보다 **큰 숫자**를 먹으면 **즉사** (목숨 1개)
- 자기 몸에 부딪혀도 즉사

### 합체 (Merge)
- 필드의 **M 아이템**을 먹으면 합체 발동
- 머리→꼬리 방향으로 스캔하며 **인접한 같은 숫자**를 greedy 매칭
- 한 패스에서 찾은 쌍을 **동시에 합체** (1+1→2, 2+2→3...)
- 변화가 없을 때까지 **반복** (연쇄 합체)
- 합체 = 점수 획득 (`기본점수 × 결과값 × 체인배수`)

### 특수 아이템
| 아이템 | 아이콘 | 효과 |
|--------|--------|------|
| Merge | ◆ M (금색 다이아몬드) | 합체 트리거 |
| Removal | ✂ (보라색 원) | 꼬리 마디 1개 잘림 |

## 먹이 시스템

### 스폰 방식
- 라운드 시작 시 **3개** 배치
- 이후 **2초마다 1개씩** 추가 (최대 **30개**)
- 시간이 지날수록 필드가 채워져 **압박 증가**

### 먹이 값 범위
- 기본: **1~3** 고정
- 머리값이 올라가면 `max(3, floor(√머리값))`으로 범위 확장

| 머리값 | 먹이 범위 |
|--------|----------|
| 1~15 | 1~3 |
| 16~24 | 1~4 |
| 25~35 | 1~5 |
| 36~48 | 1~6 |

### 카테고리 분배

먹이 생성 시 먼저 **5% 확률로 M 아이템** 체크. 나머지 95%는 아래 카테고리:

| 카테고리 | 비율 (전체 기준) | 설명 |
|----------|-----------------|------|
| **유의미** | 57% | 뱀에 이미 있는 숫자 (≤ head). 꼬리값 가중치 최대. 먹으면 합체 가능 |
| **먹을수있는** | 23.75% | ≤ head이지만 뱀에 없는 숫자. 안전하지만 합체에 즉시 쓸모없음 |
| **위험** | 9.5% | > head (head+1~3 범위). 먹으면 죽음 |
| **특수** | 4.75% | ✂ (꼬리 잘림) 또는 뱀 최소값보다 낮은 숫자 |

- 먹을수있는/위험 카테고리가 비어있으면 유의미에 합산

## 라운드 진행

### 구조
- 라운드마다 **목표 점수** 존재 (R1: 300, 이후 ×1.8 증가)
- 목표 달성 시 HUD에 **NEXT 버튼** 활성화
- NEXT를 누르면 **라운드 클리어 연출**:
  1. 머리를 제외한 **꼬리가 하나씩 터지며** 점수로 전환 (마디값 × 10)
  2. **라운드 보너스** 추가 (라운드 × 50)
  3. 1.5초 후 다음 라운드 시작
- 라운드 진행은 **선택적** — 계속 플레이해도 됨

### 속도
| 라운드 | 틱 간격 |
|--------|---------|
| 1 | 400ms |
| 2 | 370ms |
| ... | -30ms/라운드 |
| 10+ | 150ms (최대) |

## UI

### HUD (상단 2행)
- **좌측**: 라운드 번호, 점수/목표, 진행 바
- **중앙**: HEAD 값, 뱀 길이 (LEN)
- **우측**: NEXT 버튼 (목표 달성 시) 또는 꼬리 미리보기 (최근 6마디)

### 튜토리얼
- 매 게임 시작 시 HOW TO PLAY 오버레이 표시
- 이동, 먹기 규칙, M 합체, ✂ 설명
- 탭하면 dismiss → ROUND 1 ready 화면

### 오버레이 화면
| 상태 | 표시 |
|------|------|
| ready | "ROUND N" + "Press SPACE or tap to start" |
| round_clear | "ROUND N CLEAR!" + 보너스 점수 |
| game_over | "GAME OVER" + 최종 점수 |
